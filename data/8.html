17.01.2014 FFmpeg расклеивание и склеивание дорожек

ffmpeg -y -i 1.m4v -vn -map 0:2 -ss 00:13:10:500 -t 65 -vcodec copy 2.mp3

Из видео 1.m4v берем аудиодорожку начиная с 13:10:500, длительностью 65 секунд в файл 2.mp3

ffmpeg -y -i 1.m4v -an -map 0:0 -ss 789 -t 66 -vcodec copy 2.m4v

Из видео 1.m4v берем видеодорожку начиная с 789 секунды, длительностью 66 секунд в файл 2.m4v

ffmpeg -y -i 2.m4v -vcodec copy -i 2.mp3 3.m4v

Соединяем 2 файла в 3.m4v
